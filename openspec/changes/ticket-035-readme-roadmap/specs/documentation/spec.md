# README Roadmap Spec

## ADDED Requirements

### Requirement: README Project Overview

The README MUST describe the current Mapwa game concept, setup commands, development commands, and roadmap status accurately.

#### Scenario: Project overview is readable

- GIVEN a new contributor opens the README
- WHEN they read the overview
- THEN they can understand the student RPG premise and current gameplay loop

#### Scenario: Commands are accurate

- GIVEN a contributor wants to run, test, or build the project
- WHEN they follow README commands
- THEN the commands match the current repository tooling and do not conflict with AGENTS.md

#### Scenario: Roadmap is clear

- GIVEN a reader wants to know what is implemented and planned
- WHEN they read the roadmap section
- THEN implemented systems and planned roadmap areas are distinguished clearly
